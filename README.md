# Analisis-Deteksi-dan-Prediksi-Kecepatan-Mobil-Menggunakan-Haar-Cascade-Classifier
Proyek ini berfokus pada pengembangan sistem berbasis computer vision untuk mendeteksi kendaraan serta menganalisis dan memprediksi kecepatan mobil dari data video. Metode utama yang digunakan adalah Haar Cascade Classifier, yang dikenal efektif untuk deteksi objek secara real-time.
